# xfrs-lottery
🎯 零依赖、轻量级的网页抽奖解决方案。支持批量导入、自定义奖项、内定中奖人、动态背景音乐和两种抽奖模式（逐个/批量），并集成GSAP流畅动画。拥有电影级探照灯、烟花动画和可拖拽金灿灿的抽奖券。


![抽奖工具界面截图](https://imgur.com/a/9CyoyeM) 

---

## ✨ 功能特性

*   **高度可定制化**:
    *   **活动标题**: 自定义抽奖活动的主标题，并支持调整字体大小和上传自定义字体文件。
    *   **候选名单**: 批量导入所有参与抽奖的人员名单。
    *   **奖项设置**: 自由添加、删除、修改奖项名称和每个奖项的中奖名额。
    *   **背景与音乐**: 支持上传自定义背景图片和背景音乐，打造专属活动氛围。
*   **公平与趣味兼备**:
    *   **内定功能**: 可为特定奖项预设中奖人（双击奖项名称输入框进行设置），满足特殊业务需求。
    *   **随机算法**: 采用随机种子算法，保证未内定的抽奖结果公平公正。
    *   **防重复中奖**: 系统自动确保同一个人不会重复中奖。
*   **两种抽奖模式**:
    *   **逐个抽奖 (手动模式)**: 通过拖拽和点击屏幕上的虚拟抽奖券，一次抽取一名中奖者，充满悬念。
    *   **批量抽奖 (自动模式)**: 启动探照灯动画，点击停止后一次性揭晓当前奖项的所有中奖者，适合快速开奖。
*   **炫酷的视觉效果**:
    *   基于 **GSAP** 动画库，提供流畅的抽奖券拖拽、探照灯、中奖揭晓和烟花庆祝动画。
    *   富有质感的金色抽奖券和喜庆的红色主题UI。
*   **完善的结果展示**:
    *   抽奖过程中，实时在设置面板更新中奖结果。
    *   所有奖项抽取完毕后，会以精美的弹窗展示最终的完整中奖名单，并支持自动滚动播放。

## 🚀 如何使用

1.  **下载项目**: 下载本项目所有文件 (`index.html`, `gsap.min.js`, `Draggable.min.js`) 到您的电脑上。
2.  **打开页面**: 直接用 Chrome、Edge、Firefox 等现代浏览器打开 `index.html` 文件即可运行。
3.  **进行设置**:
    *   点击右下角的 **[设置]** 按钮。
    *   修改 **活动标题**。
    *   在 **候选名单** 文本框中，每行输入一个参与者的名字。
    *   在 **奖项设置** 中，设置好所有奖项的名称和数量（默认从上往下抽取）。
    *   (可选) 上传自定义的背景图片或背景音乐。
4.  **生成抽奖**: 所有设置完成后，点击 **[生成/重置抽奖]** 按钮。此时，主界面会根据候选人数量生成对应的抽奖券。
5.  **开始抽奖**:
    *   **逐个抽奖模式**: 直接用鼠标点击或拖拽界面上的任意一张抽奖券，即可开奖。
    *   **批量抽奖模式**: 点击 **[逐个抽奖]** 按钮切换到 **[批量抽奖]** 模式，然后点击 **[开始]** 启动探照灯动画，再次点击 **[停！]** 按钮即可开奖。
6.  **查看结果**: 所有奖项抽完后，会自动弹出最终中奖名单。您也可以在抽奖过程中随时进入设置面板查看实时结果。
